===============================================================================

　WCluster

===============================================================================


■ソフトの概要

　フォルダ・ファイルの圧縮・展開を行います。いわゆるアーカイバです。

　一度に圧縮できる対象は単一のフォルダ又はファイルです。
　暗号化に対応しています。

　アーカイブファイルはこのアプリ固有の形式です。


■動作環境

　Windows 7
　確認していませんが Windows 8/8.1/10 でも動くかもしれません。

　.NET Framework 4 を使用します。


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■使い方

　●圧縮

　　圧縮したいフォルダ・ファイルを "圧縮.exe" にドロップして下さい。
　　デフォルトの設定の場合、フォルダ・ファイルと同じ場所にアーカイブファイルを作成します。


　●展開

　　展開したいアーカイブファイルを "展開.exe" にドロップして下さい。
　　デフォルトの設定の場合、アーカイブファイルと同じ場所にファイル・フォルダを展開します。


　●設定

　　"圧縮.exe", "展開.exe" 又は WCluster.exe を(引数無しで)実行して下さい。


■取り扱い種別

　フリーソフト
