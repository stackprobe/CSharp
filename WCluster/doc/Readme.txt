----
コマンド

	> WCluster.exe [/F] [/P パスフレーズ] [/-] 入力パス [出力パス]

	入力パスがディレクトリの場合、ファイル（クラスタファイル）を作成します。
	入力パスがファイル（クラスタファイル）の場合、ディレクトリに展開します。

	出力パスが存在するディレクトリの場合、そのディレクトリの直下にそれなりの名前で出力します。
	出力パスを省略すると、入力パスと同じ場所にそれなりの名前で出力します。

	/F を指定すると、出力パスを上書きします。

	オプションは順不同です。/- を指定するとオプション終了と見なします。

	パスフレーズを指定すると、暗号化・復号を行います。
	復号に失敗するとエラーになります。

	非同期で実行するので、同期する場合は START /WAIT して下さい。

----
実行例

	> WCluster.exe C:\Data\0001

	> START /WAIT WCluster.exe /F /P abc123 Input\abc Output\def
